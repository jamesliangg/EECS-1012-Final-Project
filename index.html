Project Description

Our team is working on developing an engaging web-based game that challenges users to guess a number between 1 and 10. Upon launching the application, players can enter a display name to personalize their gaming experience. The game will provide the user with nine lives, and they must use their deduction skills to guess the number selected by the server. The user will earn a point for every correct guess, while a wrong guess will result in a deduction of one life. In case of an incorrect guess, the computer will provide a hint indicating whether the number is higher or lower. Once the user guesses the number correctly, they can continue the game with a new random number. Users' remaining lives will carry forward to the next attempt. However, if they exhaust all their lives, the game will display their score and offer them the option to restart with zero points and nine lives.
